UEFI Advanced Mode

The ASUS Advanced Mode Main menu screen appears when you enter Advanced Mode. As shown in Figure 1, this mode provides options to configure advanced UEFI settings.

Note: To return to EZ Mode, press F7.

These are the items found on the menu on top of the screen:

  • My Favorites – (Figure 2) Used to provide quick access to frequently-used settings. Press F3 or click My Favorites (F3) to open the Setup Tree Map Screen shown in Figure 3.

  • Main – (Figure 4) Used to display computer information and change the basic configuration. It allows you to set the date, time, language, and security settings.

  • Ai Tweaker – (Figure 5) Used for changing the CPU overclocking settings and memory operating settings. Always be cautious when changing these settings. Incorrect settings could cause the system to malfunction.

  • Advanced - (Figure 6) Used to change the settings for the CPU and other devices. Always be cautious when changing these settings. Incorrect settings could cause the system to malfunction.

  • Monitor - (Figure 7) Used for displaying temperatures, power status, and changing fan speed settings.

  • Boot - (Figure 8) Used for changing the boot options.

  • Tool - (Figure 9) Used for configuring options for special functions.

  • Exit - (Figure 10) Used for selecting the exit options and loading default settings.
