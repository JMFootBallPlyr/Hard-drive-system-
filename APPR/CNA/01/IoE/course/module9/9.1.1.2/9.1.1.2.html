Common Input Devices and LEDs in Laptops

Laptops are designed to be compact and portable, while maintaining much of the same functionality provided by desktop computers. As a result, essential input devices are built into laptops. When a laptop lid is lifted, the following input devices may be present:

  • Touchpad

  • Pointing stick

  • Keyboard

  • Fingerprint readers

  • Microphone

  • Web camera

Note: Input devices that are built into laptops can be configured in the same manner as input devices for desktops.
