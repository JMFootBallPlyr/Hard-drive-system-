Local Password Management

Password management for stand-alone Windows computers is done locally through the User Accounts tool. To create, remove, or modify a password in Windows, use the following path, as shown in Figure 1:

Control Panel > User Accounts

To prevent unauthorized users from accessing local computers and network resources, lock your workstation, laptop, or server when you are not present.

It is important to make sure that computers are secure when users are away. A security policy should contain a rule about requiring a computer to lock when the screensaver starts. This will ensure that after a short time away from the computer, the screen saver will start and then the computer cannot be used until the user logs in.

In all versions of Windows, use the following path:

Control Panel > Personalization > Screen Saver. Choose a screen saver and a wait time, and then select the On resume, display logon screen option, as shown in Figure 2.
